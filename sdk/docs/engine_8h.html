◆ NGEngineInit()

void NGEngineInit ( void )

Calls: NGArenaSystemInit, NGPalInitDefault, NGPalInitAssets, NGFixClearAll, NGSceneInit, NGCameraInit, NGInputInit, NGAudioInit Also sets backdrop color to black.

NGPalInitAssets() is called automatically - if the game includes ngres_generated_assets.h, its palettes will be loaded; otherwise a weak no-op default is used.
