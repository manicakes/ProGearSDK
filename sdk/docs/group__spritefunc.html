◆ NGSpriteScreenToHardwareY()

s16 NGSpriteScreenToHardwareY ( s16 screen_y)
inlinestatic

NeoGeo sprites use inverted Y: 496 at screen top, decreasing downward. Values wrap at 512 (9-bit field).

Parameters
screen_yScreen Y coordinate (0 = top of screen)
Returns
Hardware Y value (9-bit, 0-511)

u8 NGSpriteAdjustedHeight ( u8 rows,
u8 v_shrink )
inlinestatic

When sprites are shrunk vertically, fewer tile rows are visible. This calculates the actual number of rows needed to fill the display area without garbage pixels.

Parameters
rowsOriginal height in tiles (1-32)
v_shrinkVertical shrink value (0-255, 255 = full size)
Returns
Adjusted height in tiles (1-32)
