7// ui.h - UI system for menus and dialogs
8//
9// The UI system provides sprite-based, animated interface elements.
10// UI elements are positioned in VIEWPORT coordinates (screen-relative),
11// not world coordinates. They appear at a high Z-index, always on top.
12//
13// The system uses spring physics for smooth, modern-feeling animations:
14// - Menus slide in/out with spring motion
15// - Selection cursor animates between items with overshoot
16// - Background dimming via palette fade (no extra sprites!)
17// - All animations are snappy (~200-300ms)
18
