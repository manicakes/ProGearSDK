Detailed Description

The fix layer is a 40x32 tile layer used primarily for text and HUD. It overlays sprites and is not affected by scrolling.

Screen Layout

  • Total: 40x32 tiles (640x512 virtual)
  • Visible: 40x28 tiles (320x224 pixels)
  • Safe area: 38x25 tiles (accounts for CRT overscan)
